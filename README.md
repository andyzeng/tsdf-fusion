# KinFu
Integrate depth frames into a fused projective TSDF volume using GPU (with CUDA)

## Dependencies
1. PNG++, a lightweight library for PNG image I/O
 * `sudo apt-get install libpng++-dev`
2. NVIDIA graphics card and CUDA

## Compilation
`./compile`

## Sample Usage
`./kinfu`
